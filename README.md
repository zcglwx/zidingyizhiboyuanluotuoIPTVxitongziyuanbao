## 安装步骤

1. **环境准备**: 建议使用宝塔面板进行安装。
2. **创建网站**: 在宝塔面板中创建一个新的网站，并将后端源代码上传到网站根目录。
3. **创建数据库**: 在宝塔面板中创建一个新的数据库，并将数据库文件导入。
4. **配置数据库连接**: 修改`conn.php`文件中的数据库信息，确保与您创建的数据库信息一致。
5. **访问后台**: 打开域名/sysadmin.php，进入后台管理页面。默认账号和密码均为`admin`。
